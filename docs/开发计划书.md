# 消费记录管理 App 开发计划书

> **版本**: v1.0
> **创建日期**: 2024-01-15
> **最后更新**: 2026-04-25

## 1. 项目概述

- **项目名称**: Wallet
- **项目类型**: Web 单页应用（HTML + CSS + JavaScript）
- **核心功能**: 在浏览器中记录消费数据，生成周期报表和趋势图表
- **目标用户**: 需要管理日常消费的个人用户

---

## 2. 功能需求

### 2.1 消费记录管理

| 功能 | 描述 |
|------|------|
| 添加消费 | 记录日期、金额、类别、子类别、备注 |
| 编辑消费 | 修改已有记录（点击首页记录进入编辑） |
| 删除消费 | 删除单条记录 |
| 消费列表 | 按时间倒序展示 |
| CSV 同步 | 每次操作自动同步到本地 CSV 文件 |

### 2.2 消费类别与子类别

| 类别 | 子类别 |
|------|--------|
| 餐饮 | 早餐、午餐、晚餐、零食、饮料 |
| 交通 | 打车、公交、地铁、加油、保险、停车 |
| 购物 | 日用品、网购、服饰、数码 |
| 居住 | 房租、水电、物业、装修 |
| 娱乐 | 电影、游戏、旅游、运动 |
| 医疗 | 门诊、买药、体检 |
| 其他 | 礼品、捐赠、其它 |

### 2.3 子类别管理

- 每个主类别有预置子类别
- 用户可自定义添加新子类别
- 子类别存储在 localStorage 中，随 CSV 同步

### 2.4 报表功能

| 报表类型 | 内容 |
|----------|------|
| 周报 | 本周消费总额、每日消费堆叠柱状图、与上周对比 |
| 月报 | 本月消费总额、每日消费堆叠柱状图、与上月对比 |
| 年报 | 本年度消费总额、每月消费堆叠柱状图、与去年对比 |
| 总报 | 最早到最晚消费、按月/按年颗粒度可选 |

### 2.5 图表功能

| 图表类型 | 描述 |
|----------|------|
| 堆叠柱状图 | 展示周期内每日/每月消费，类别堆叠显示 |
| 饼图 | 展示类别占比，点击可 drill-down 到子类别 |
| 柱状图/饼图切换 | 周报/月报/年报/总报均支持 |
| 饼图 drill-down | 点击饼图扇区进入该类别，查看子类占比 |

### 2.6 趋势分析

- 消费趋势图（折线图，支持周/月/年切换）
- 类别占比图（环形饼图）
- 月预算设置与进度显示

## 3. 技术方案

### 3.1 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Vue 3（CDN引入）|
| 样式 | 自定义工具类（Tailwind 风格）|
| 图表库 | ECharts 5（CDN引入）|
| 本地存储 | localStorage（内存）+ CSV 文件（持久化）|
| 日期处理 | Day.js（CDN引入）|

### 3.2 数据存储

#### CSV 文件格式
```
id,amount,category,subcategory,date,note,createdAt,updatedAt
uuid-1,25.50,餐饮,午餐,2024-01-15,工作餐,1705300000000,1705300000000
```
- **编码**: UTF-8 (BOM 可选，兼容 Excel)
- **换行符**: CRLF (Windows 兼容)

#### 存储策略
1. **主要存储**: localStorage（快速读写）
2. **持久化**: CSV 文件（用户授权后自动同步）
3. **File System Access API**: Chrome/Edge 支持直接读写本地文件
4. **备选方案**: 手动导出/导入 CSV

### 3.3 数据模型

```javascript
// 消费记录 (Expense)
{
  id: string,           // UUID
  amount: number,       // 金额（单位：元）
  category: string,     // 主类别
  subcategory: string,  // 子类别
  date: string,         // 日期 (YYYY-MM-DD)
  note: string,         // 备注（可选）
  createdAt: number,    // 创建时间戳
  updatedAt: number     // 更新时间戳
}

// 子类别配置 (SubCategories)
{
  "餐饮": ["早餐", "午餐", "晚餐", "零食", "饮料"],
  "交通": ["打车", "公交", "地铁", "加油", "保险", "停车"],
  ...
}
```
